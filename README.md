# Convert-to-Solr-Xml-Format-
This is a simple xml-parser project.The goal is to be able to index the xml file in the desired format to solr.
Which is like this: 
<add>
  <doc>
    <field name="...">.....</field>
    <field name="...">.....</field>
    <field name="...">.....</field>
  </doc>
  <doc>
    <field name="...">.....</field>
    <field name="...">.....</field>
    <field name="...">.....</field>
  </doc>
</add>
